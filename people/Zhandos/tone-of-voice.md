# tone-of-voice.md — Мой стиль письма

> Как **я лично** пишу: в Slack, в комментариях на ревью, в внутренних документах, в письмах.
> Не путать с `/team/brand-tone.md` — там о том, как пишет продукт для пользователя.
>
> Самый ценный сигнал — это **твои примеры текстов** (раздел ниже). Не пропускай.

## Базовые характеристики

**Регистр в рабочей коммуникации:**

- [ ] Формальный (отчёты, презентации руководству)
- [x] Полуформальный (продуктовые обсуждения, документация)
- [ ] Разговорный (Slack, переписка с командой)
- [ ] Меняется в зависимости от: _____________

**Обращение:**

- [ ] На «ты» — со всеми
- [x] На «ты» — с командой, на «вы» — с клиентами / стейкхолдерами
- [ ] На «вы» — со всеми

**Длина высказывания:**

- [ ] Короткая, по делу
- [x] Развёрнутая, с контекстом
- [ ] Зависит от:

**Юмор:**

- [ ] Использую регулярно
- [x] Редко, точечно
- [ ] Не использую в рабочих текстах

**Эмодзи:**

- [ ] Активно (Slack, Notion)
- [ ] Точечно
- [x] Никогда в рабочем
