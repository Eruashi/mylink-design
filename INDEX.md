# Designer Skills Index
> Персональная коллекция скиллов для дизайнера.  
> Загружай нужные категории или отдельные скиллы через raw-ссылки ниже.

---

## Как использовать

**Загрузить один скилл**: скинь raw-ссылку в чат и попроси меня прочитать её.

**Загрузить категорию**: скинь несколько ссылок из одного раздела.

**Project Instructions** (рекомендуется): вставь в инструкции проекта нужные ссылки — скиллы будут активны в каждой сессии автоматически.
