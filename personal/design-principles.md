## Обязательные стандарты (без них не отдаю работу)

- [ ] Контраст WCAG AA минимум для всех текстов
- [ ] Touch targets ≥ 44pt на mobile
- [ ] Все интерактивные состояния: hover, focus, active, disabled, loading, error
- [ ] Empty states для всех data-driven экранов
- [ ] Loading skeletons / shimmer для ожидания ≥ 500ms
