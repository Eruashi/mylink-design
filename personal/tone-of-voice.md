# tone-of-voice.md — Твой стиль

> Как ты говоришь и пишешь. LLM подстраивается под этот файл при генерации текстов: deliverables, описаний, ответов, комментариев на ревью.
> Самый ценный сигнал — это твои примеры текстов (раздел ниже). Не пропускай.

---

## Базовые характеристики

**Регистр:**

- [ ] Формальный (отчёты, презентации руководству)
- [ ] Полуформальный (продуктовые обсуждения, документация)
- [ ] Разговорный (Slack, переписка с командой)
- [ ] Меняется в зависимости от: _____________

**Обращение:**

- [ ] На «ты» — со всеми
- [ ] На «ты» — с командой, на «вы» — с клиентами / стейкхолдерами
- [ ] На «вы» — со всеми
- [ ] В продуктовых текстах для пользователя: _(ты / вы / без обращения)_

**Длина высказывания:**

- [ ] Короткая, по делу
- [ ] Развёрнутая, с контекстом
- [ ] Зависит от:

**Юмор:**

- [ ] Использую регулярно
- [ ] Редко, точечно
- [ ] Не использую в рабочих текстах

**Эмодзи:**

- [ ] Активно
- [ ] Точечно (Slack, неформальное)
- [ ] Никогда в продуктовых текстах
- [ ] Никогда вообще

> Перечисли табу. LLM будет избегать.

**Слова и выражения:**

- _(например: «синергия», «бесшовно», «эффективно», «давайте погрузимся»)_

**Стилистика:**

- _(например: канцелярит, корпоративный шаблон, новостной слог, кликбейт)_

- _(например: длинные сложноподчинённые с тремя «который»; пассив, когда можно актив)_

## Anti-tone — как я НЕ хочу звучать

> «Не как корпоративный шаблон», «не как новостник», «не как блогер с эмодзи».
> Будь конкретен — это сильнее ограничивает LLM, чем позитивные критерии.
